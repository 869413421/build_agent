# 北极星目标

## 使命

从 0 到 1 打造一个教程优先、工程可落地的 Agent 框架，并将其应用到劳动纠纷处理指导智能体。

## 成功标准

1. 新用户可在 60 分钟内跑通 MVP。
2. 框架明确覆盖五项核心能力：
   - 记忆
   - 检索
   - 上下文工程
   - 智能体通信协议
   - 智能体评估
3. 劳动纠纷智能体可输出：
   - 纠纷处理路径建议
   - 证据清单
   - 文书草稿建议
4. 涉及检索的结论必须附可追溯引用，且输出必须带安全免责声明。

## 非目标（v1）

1. 不替代持证律师服务。
2. 不提供结果保证或胜诉概率承诺。
3. v1 不覆盖多法域全球法律体系。

## 主要受众

希望学习并实践 Agent 工程化落地的后端工程师。
