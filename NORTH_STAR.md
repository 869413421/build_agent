# 北极星目标

## 使命

从 0 到 1 打造一个教程优先、工程可落地的 Agent 框架，并将其应用到劳动纠纷处理指导智能体。

## 成功标准

1. 新用户可在 60 分钟内跑通最小可用版本（MVP）。
2. 框架能力按顺序落地：协议、执行、可观测、上下文、检索、记忆、评测、安全。
3. 劳动纠纷智能体可输出：处理路径、证据清单、文书草稿建议。
4. 涉及检索的结论必须附可追溯引用，所有法律输出必须附免责声明。

## 执行优先级（强制顺序）

1. Protocol
2. Engine（loop）
3. Model Runtime（LLM Adapter）
4. Tool Runtime（API Adapter）
5. Observability
6. Context Engineering
7. Retrieval
8. Memory
9. Evaluator
10. Safety Layer

## 学习路径约束

1. 每次只实现一个组件。
2. 每个组件必须有代码、测试、教程三件套。
3. 每步完成后必须等待用户审核通过，再进入下一步。

## 每日输出约束

1. 每天至少一篇教程文章。
2. 当天文章必须对应当天实际代码变更。
3. 教程必须可复现，有命令、有结果、有验证。

## 非目标（v1）

1. 不替代持证律师服务。
2. 不提供胜诉概率承诺或结果保证。
3. 不在 v1 覆盖多法域全球法律体系。

## 会话交接要求

每次会话结束前必须更新 `PROJECT_STATUS.md`，保证下个会话或其他 AI 可直接接手。
