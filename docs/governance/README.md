# Governance 文档索引

为便于维护，所有项目治理与会话交接文档统一放在本目录。

## 文件说明

1. `NORTH_STAR.md`：项目北极星与长期目标
2. `AI_TASK_GUARDRAILS.md`：AI 执行硬约束
3. `PROJECT_STATUS.md`：当前进度与下一步唯一任务
4. `ROADMAP_COMPONENTS.md`：组件顺序与 DoD
5. `HANDOFF_CHECKLIST.md`：会话结束交接清单

## 使用顺序（新会话）

1. 先读 `NORTH_STAR.md`
2. 再读 `AI_TASK_GUARDRAILS.md`
3. 再读 `PROJECT_STATUS.md`
4. 最后按 `ROADMAP_COMPONENTS.md` 执行
